This bot can give you information about the valorant agents.









Prefix:

V


Description:

The Valorant Wiki can give you information about your favourite agents in the game.


Commands:

- V agents type this to get all the names of the agents.

- V agent {AGENT_NUMBER} To get more info about a agent.

- V weapons type this to get all the names of the weapons.

- V weapon {WEAPON_NUMBER} To get more info about a weapon.

- V maps type this to get all the names of the agents.

- V map {MAP_NUMBER} To get more info about a map.

- V agents type this to get all the names of the agents.

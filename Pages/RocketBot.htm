Rocket Launch Tracker

This bot will notify the users in your server when a new rocket launch is announced.









Prefix:

RT


Description:

The Rocket Launch Tracker will get the latest planned rocket launches and it will send a message in your channel when a new launch was added.


Commands:

- RT track to get info about the recentlyadded launch

- RT get [number from 1 to 5] to get the latest added launches

- RT start type this command to start the notifications in your channel (only for users with admin permissions)

- RT stop use this command to stop the notifications in your server. (only for users with admin permissions)

- RT help type this command to get all the commands you can use

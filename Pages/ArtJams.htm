Weekly Art Jams Bot

This bot will send a random theme every Monday at 12:00 CEST.









Prefix:

TB


Description:

This bot will send a random theme every Monday at 12:00 CEST.

It will send a theme out of a list with more than 100+ themes.


Commands:

- Start the bot with TB start (only for users with admin permissions).

- Stop the bot with TB stop (only for users with admin permissions).

- Get a theme TB theme (you can use this command for maybe if you are bored and want to draw something. You will get a random theme).
